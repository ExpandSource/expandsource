# PRD (Product Requirements Document)

## 프로젝트 개요

**프로젝트명:** ExpandSource Lab
**타입:** 개인 포트폴리오 / GitHub Pages
**상태:** 진행 중
**최종 목표:** 개인 브랜드 및 프로젝트 통합 관리 페이지

---

## 1. 프로젝트 개요

ExpandSource Lab은 개인 웹페이지로서, 개발자의 진입점 역할을 하는 정적 페이지입니다. 향후 다양한 프로젝트와 개인 소개 콘텐츠를 확장할 수 있는 기초 플랫폼입니다.

---

## 2. 목표 및 목적

### 주요 목표
- 개인 브랜드의 중심 허브 구축
- 프로젝트 및 경력 정보의 단일 진입점 제공
- 모던하고 전문적인 이미지 표현

### 세부 목표
- 기본적인 HTML/CSS로 빠른 구현
- 반응형 디자인으로 모든 디바이스 지원
- 향후 확장 가능한 아키텍처

## 4. 디자인 사양

### 색상 팔레트
- **배경:** 검은색 계통 그라데이션 (`#1a1a1a` → `#2d2d2d`)
- **텍스트:** 회색 계통 그라데이션 (`#d0d0d0` → `#707070`)
- **액센트:** 회색 톤 (`#555` ~ `#999`)
- **테마:** 다크 모드

### 레이아웃
- **구조:** 중앙 정렬, 반응형
- **헤더:** 페이지 중앙에 제목/부제
- **Footer:** 하단 고정, 링크 포함
- **Breakpoints:**
  - Desktop: 1024px+
  - Tablet: 768px ~ 1023px
  - Mobile: 480px ~ 767px

### 타이포그래피
- **폰트:** System fonts (`-apple-system`, `BlinkMacSystemFont`, `Segoe UI`, etc.)
- **제목:** 2.5em, 그라데이션 적용
- **부제:** 1.2em, 회색톤
- **본문:** 1em, 회색톤
